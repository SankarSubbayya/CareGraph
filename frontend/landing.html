Graph Intelligence

Healthcare is about relationships

Traditional databases store records. Neo4j stores connections — and connections save lives.

Dorothy —[TAKES]→ Lisinopril 10mg
Dorothy —[REPORTED]→ dizziness
Lisinopril 10mg —[SIDE_EFFECT]→ dizziness
dizziness —[SUGGESTS]→ Hypotension
Graph Insight: Dorothy's dizziness may be a side effect of Lisinopril. This was detected automatically by traversing the graph — not by keyword matching. Recommend discussing with her doctor.
Technology

Built with purpose

Every technology chosen to serve a specific role in the care pipeline.

Graph Database
Neo4j Aura

Models the care network: seniors, medications, symptoms, conditions, family. 9 node types, 9 relationship types. Cypher queries detect drug interactions and side effects in milliseconds.

AI Pipelines
RocketRide AI

Visual pipeline orchestration. Four .pipe files route prompts through Gemini LLM for transcript analysis, drug explanations, care plans, and condition suggestions.

Voice Agent
Bland AI

Automated phone calls with warm, patient AI voices. Asks about mood, medications, symptoms. Transcripts arrive via webhook for instant processing.

LLM Inference
GMI Cloud

DeepSeek-R1 via OpenAI-compatible API. Powers CrewAI agents and serves as fallback when RocketRide is unavailable.

Agent Orchestration
CrewAI

5 specialized agents collaborate: Check-in, Analysis, Graph, Recommendation, and Alert. Sequential pipeline ensures thorough care analysis.

Backend + Frontend
FastAPI + vis.js

Python REST API with 34 endpoints. Interactive graph visualization. Real-time dashboard for families and care providers.
